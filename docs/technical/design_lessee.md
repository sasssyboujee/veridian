# Veridian Lessee Portal Design System

## 1. Brand Identity
**Theme:** "Clean Consumer Energy"
Unlike the high-tech dark mode of the Operations portals, the Lessee portal uses an approachable, light-themed, minimalist UI. It focuses on clarity, readability, and immediate comprehension of savings and hardware status.

## 2. Color Palette (Light Mode)
- **Primary / Active:** `#76B900` (Veridian Green)
- **Primary Background:** `#ffffff` (White)
- **Card / Surface Background:** `#f9fafb` (Off-white)
- **Primary Text:** `#1a1a1a` (Near Black)
- **Subtext / Secondary Text:** `#6b7280` (Gray 500)
- **Label Text:** `#374151` (Gray 700)
- **Borders & Dividers:** `#e5e7eb` (Gray 200) / `#d1d5db` (Gray 300)
- **Success:** `#15803d` (Green 700)

## 3. Typography
- **Primary Font:** `system-ui, -apple-system, sans-serif`
- **Branding / Numbers (Selective):** `Roboto Mono` (Used sparingly for the logo mark or specific metrics).
- **Styling:** Friendly, rounded sans-serifs with normal to tight letter spacing.

## 4. UI Elements
- **Radii:** `16px` for main cards (soft and friendly) and `8px` for inputs/forms.
- **Borders:** Subtle `1px solid #e5e7eb` on cards and headers.
- **Forms:** Inputs use a white background `#ffffff` with a light gray border (`#d1d5db`), changing border color on focus.
- **Layout:** Centered, single-column maximal reading width (`800px`) for focus and simplicity.
