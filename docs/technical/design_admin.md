# Veridian Admin & Operations Design System

## 1. Brand Identity
**Theme:** "Veridian Capital (Deep Learning High-Tech)"
The admin portals merge the solidity of traditional finance with the high-tech aesthetics of decentralized infrastructure.

## 2. Color Palette (Dark Mode)
- **Primary / Active:** `#008080` (Deep Teal / Veridian Green)
- **Primary Logo Accent:** `#76B900` (Neon Green)
- **Obsidian (Base Background):** `#121212`
- **Deep Charcoal (Secondary Base/Hero):** `#1C1C1C`
- **Dark Neutral (Borders & Dividers):** `#2E3033`
- **Pale Sage (Accents):** `#E2ECE9`
- **White (Primary Text):** `#FFFFFF`
- **Success:** `#6EFA5F`
- **Error:** `#CF202F`

## 3. Typography
- **Primary Tech Font:** `Roboto Mono, monospace` (Used for telemetry, numbers, grids, and primary action buttons).
- **Secondary Display:** `Söhne / SF Pro Display / Inter` (Used for marketing-style hero headers).

## 4. UI Elements
- **Radii:** `8px` (`--rounded-base`) for structural cards and inputs.
- **Effects:** Active elements often have a teal text-shadow glow.
- **Borders:** Cards use a "tech border" style (`3px solid var(--color-primary)`) on the left edge to denote active data blocks.
